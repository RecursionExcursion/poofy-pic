# 🪄 Poofy Pic — Custom Chrome Extension

### 🖼️ Setup Instructions:

1. Clone or unzip this folder
2. Drop your `.jpeg`, `.png`, or `.webp` files into the `/assets` folder
3. Open `picker.html` in Chrome
4. Click "Generate List" to create a `poofy_list.json`
5. Move that file into `/assets/`
6. Go to `chrome://extensions` and click "Reload" on this extension
7. Visit any site — your images will auto-poof 50% of images on the page!

✅ Works until you disable or remove the extension  
🔁 To update images, repeat steps 2–6
